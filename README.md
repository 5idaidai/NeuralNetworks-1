NeuralNetworks
==============

Implementations of several types of Neural Networks

*   models/autoencoder.py : sparce autoencoder
*   models/denoising_autoencoder.py: denoising autoencoder 
*   models/mlp.py: multi-layer perceptron
*   models/stacked_autoencoder.py: stacked denoising (sparse) autoencoder
*   models/conv_mlp.py: convolutional multi-layer perceptron
